# Rose Noir — Prism Launcher theme

Theme generated from a custom palette (rose / charcoal).

## Installation
1. Copy the `Rose Noir` folder into your Prism Launcher themes directory:
   - Windows: `%appdata%\PrismLauncher\themes`
   - Linux: `~/.local/share/PrismLauncher/themes`
   - macOS: `~/Library/Application Support/PrismLauncher/themes`
2. Restart Prism Launcher.
3. Settings → Launcher → User Interface → Widgets → choose **Rose Noir**.
